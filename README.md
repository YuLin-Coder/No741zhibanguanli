## 项目介绍
基于SSM框架实现一个值班管理系统, 并基于Shiro实现一个基于角色的访问控制系统, 该值班系统能够对用户, 角色, 角色权限, 值班啊信息进行管理, 并能导出excel列表, 还能查看历史操作

实现功能
该项目实现了一个带多角色访问控制的值班管理系统, 该系统实现了如下功能
分页查询值班列表功能
值班信息的增加, 删除, 修改
多角色的登录退出功能
系统管理员能够对角色进行修改, 能够对角色分配权限
系统能够批量导出excel报表功能
